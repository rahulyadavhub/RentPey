# RentPey 🏠

WhatsApp-based rent management bot for Indian landlords.

## Features
- Monthly reminders to tenants automatically
- Payment screenshot verification flow
- Owner confirmation before receipt
- Auto receipt generation
- Late payment nudges

## Tech Stack
- Node.js + Express
- MongoDB Atlas
- Meta WhatsApp Cloud API
- Render.com
